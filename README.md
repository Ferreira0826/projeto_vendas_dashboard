
# 📊 Dashboard de Vendas — Pipeline de Dados

Projeto completo de Engenharia de Dados com ETL, API e Dashboard interativo.

## 🚀 Tecnologias

- Python (ETL)
- Django + Django REST Framework (API)
- SQLite (banco de dados)
- React + Vite (frontend)
- Recharts (visualização)

---

## 🔄 Arquitetura

ETL → Banco → API → Frontend

---

## 📊 Funcionalidades

- KPIs de vendas (receita, ticket médio, crescimento)
- Forecast de receita
- Ranking de vendedores
- Análise por categoria e canal
- Filtros dinâmicos
